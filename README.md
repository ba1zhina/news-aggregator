# Новостной агрегатор

Веб-приложение для отображения и фильтрации новостей с возможностью фильтрации по тональности и категориям.

## Технологии

- Backend: Node.js (Express)
- Frontend: HTML, CSS, JavaScript
- База данных: MongoDB (в Docker)

## Установка и запуск

1. Установите зависимости:
```bash
npm install
```

2. Запустите MongoDB в Docker:
```bash
docker-compose up -d
```

3. Запустите сервер:
```bash
npm start
```

Приложение будет доступно по адресу: http://localhost:3000

## Функциональность

- Отображение списка новостей, отсортированных по дате
- Фильтрация по тональности (позитивные, нейтральные, негативные)
- Фильтрация по категориям
- Возможность выбора нескольких фильтров одновременно
- Модальное окно с полной информацией о новости
- Сортировка новостей по тональности при выборе нескольких тональностей

## Структура данных в MongoDB

Каждый документ в коллекции `news` содержит следующие поля:
- `title`: Заголовок новости
- `text`: Полный текст новости
- `category`: Категория новости
- `tone`: Тональность (positive, neutral, negative)
- `url`: Ссылка на источник
- `date`: Дата публикации
